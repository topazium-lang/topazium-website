Topazium Lang

What is Topazium?

Topazium is an language based on ruby with sinatra focused on help new people to make websites easy....

Why i call it a lang?

I call it a lang because it have custom functions based on html but more simple

Example:

require 'require_all'
require_all 'prefix/'
require 'sinatra'
get '/' do
## |
## V Your code goes here
Top::start +
Top::heo +
Top::tio + "Test" + Top::tic +
Top::hec +
Top::bo +
Top::ceo +
Top::h1o + "Hello world" + Top::h1c +
Top::po + "How are u?" + Top::pc +
Top::cec +
Top::bc
##
end

Installation:

step 1 - Clone git clone https://github.com/topazium-lang/topazium ~/topazium
* the ~/topazium part is required
step 2 - Install cd ~/topazium ./install

Usage

Create a project: topazium -s or topazium -r (s for sinatra dn r for ramaze)
Execute an project: topazium -p
